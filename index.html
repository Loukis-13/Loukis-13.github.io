Timestamp Microservice


API para converter datas em epochs e epochs em datas

End-points: GET /api/timestamp --> retorna a data atual GET /api/timestamp/{data} --> retorna a data passada convertida em epochs e utc

Request Header Parser Microservice


API para o formatar o header do requerimento HTTP

End-points: GET /api/header-parser/whoami --> retorna o IP, a lingua do sistema e o software do úsuario que fez a chamada

URL Shortener


API para encurtar URLs e permitir que sejam acessadas a partir de um índice

End-points: POST /api/urlshortener/shorturl --> verifica se a url é valida e a salva no banco de dados GET /api/urlshortener/shorturl/{short} --> acessa uma url de acordo com seu índice no banco de dados

Exercise Tracker


API para criar registros de úsuarios e associar informações de exercícios a eles

End-points: POST /api/exercise-tracker/users POST /api/exercise-tracker/users/:_id/exercises GET /api/exercise-tracker/users GET /api/exercise-tracker/users/:_id/logs

File Metadata


API para os metadados de um arquivo

End-points: POST /api/filemetadata/fileanalyse --> retorna o nome, tipo e tamnho do arquivo enviado (os arquivos enviados não serão salvos)
